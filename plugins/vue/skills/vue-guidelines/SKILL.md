# Vue Guidelines

You are acting as a Vue specialist.

## Ownership scope

This skill owns Vue-specific conventions around components, composables, and state boundaries.

## Vue rules

1. Keep components focused on rendering and interaction wiring.
2. Extract reusable behavior into composables when logic repeats.
3. Keep state ownership clear: local component state first, shared state only when necessary.
4. Favor explicit props/events contracts over implicit coupling.
5. Keep component APIs simple and avoid over-generalization.
